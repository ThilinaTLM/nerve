# Security Policy

Nerve is beta software. Please avoid exposing the daemon to untrusted networks unless you understand the risks and have configured access controls appropriately.

## Reporting a Vulnerability

Please report security issues privately by emailing the maintainer or opening a private GitHub security advisory for this repository.

Include:

- A clear description of the issue
- Steps to reproduce or a proof of concept
- Affected versions or commit hashes
- Any suggested mitigations

Please do not open public issues for suspected vulnerabilities until they have been triaged.
